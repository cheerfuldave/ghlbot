A modern chatbot solution for querying your WordPress database with natural language processing.

== Description ==

Ask Your Database WP Chatbot allows your website visitors to query your database using natural language. Built with modern WordPress standards and a beautiful user interface.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ask-your-database-wp-chatbot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Ask Your Database screen to configure the plugin
